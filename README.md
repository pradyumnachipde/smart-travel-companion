# Smart Travel Companion

A full-stack MERN travel planning application that helps users:

- Plan trips
- Manage itineraries
- Track expenses
- Maintain travel journals
- Explore destinations on an interactive map

## Tech Stack

Frontend:
- React
- Tailwind CSS
- React Router

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Maps:
- React Leaflet
- OpenStreetMap
